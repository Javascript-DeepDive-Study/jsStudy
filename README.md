## 스터디 전 준비 사항

- 책 읽고 공부하기
- 회차 폴더 안에 {이름}.md 파일 안에 공유할 내용 정리하기
  - 공부한 내용 정리
  - 질문사항, 관련 예제 
  - 토론 주제 적어오기
  - 인상 깊은 점
  - 새로 알게된 개념
  - 실습 과제(문제) 만들어오기

## 📚 스터디 진행방식

- 룰렛 돌려서 발표자 2명 선정, 진행
- 3장 발표하면, 50%씩 나눠서 발표 진행
- 발표자가 먼저 공유할 내용(질문) 발표
- 각자 적어온 질문 공유, 답변해보기
- 공부해 온 주제에 대해 자유롭게 토론, 인사이트 공유
- 정적이 흐르면 발표자가 토론 유도


## 스터디 규칙

- 불참시 레드카드 1장 = 벌금 5천원 (레드카드 3장 시 스터디방 내보내기)
- 옐로카드 2장 = 벌금 5천원
- 지각은 10분까지 인정. 이후에 들어오면 옐로카드 1장

- 진행자로 선정되었는데 준비를 안해왔으면 옐로카드 1장
