## 📚 스터디 진행방식

- 룰렛 돌려서 발표자 선정, 진행
- 진행자가 먼저 공유할 내용(질문) 발표
- 각자 적어온 질문 공유, 답변해보기
- 공부해 온 주제에 대해 자유롭게 토론, 인사이트 공유
- 정적이 흐르면 진행자가 토론 유도
